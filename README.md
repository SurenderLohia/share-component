# Share component

### How to run

#### - Clone the repo

#### - Navigate to the cloned repo

### Install dependency

In terminal, in project root directory run below command

```
yarn install
```

### To run development mode

```
yarn run dev
```

### View the storybook in browser

In browser navigate to the below url

```
http://localhost:8080/
```

### Tech Stack

- HTML, CSS, JavaScript
- React, Typescript
- Tailwindcss
